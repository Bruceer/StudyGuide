
# StudyGuide
本表意图以清华给出的《电子信息科技知识体系的MAP》为指导，  
罗列电子类相关专业课程，快速构建知识框架体系，尽早预习各类课程。  
不保证所有课程质量最优，但希望能广开思路。  
本列表中课程将竭尽全力无死角覆盖EECS所有成熟课程  
学习顺序可以是：本校要求—>名校要求—>自身兴趣，祝各位学习愉快！  
ReadMe文件与excel文件基本相同，供快速选择
